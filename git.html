Git Cheat Sheet

git config

编辑配置文件
git config [--global] -e
git config -l
输出彩色信息
git config color.ui true
不忽略大小写
git config core.ignoreCase false
git config push.default simple
git config

git log

显示提交日志
git log
git log -p <file>
以图表形式显示提交日志
git log --graph --all --oneline --decorate
显示每次提交的对象信息
git log --pretty=raw
显示指定作者的提交信息
git log --author=<author>
根据提交信息中的关键字查找
git log --grep=<keyword>
git log master...develop
显示合并提交与根提交信息
git log --first-parent
git shortlog
git shortlog -sn

git status

查看工作区的信息
git status
查看工作区信息并显示分支及追踪信息
git status -sb
并查看忽略文件的信息
git status --ignored
列出忽略文件
git check-ignore *

git remote

添加远程仓库并命名为 origin
git remote add origin <git-repo>
修改远程仓库的地址
git remote set-url origin <git-repo>
列出所有的远程仓库
git remote -v
列出远程仓库 origin 的详细信息
git remote show origin
推送 master 到远程仓库
git push origin master
推送到远程仓库并建立追踪关系
git push -u origin master
推送时忽略 git hooks
git push -n

git branch

列出本地分支
git branch
列出本地分支与追踪关系
git branch -vv
列出远程分支
git branch -r
列出所有分支
git branch -a
建立分支不切换工作区
git branch <branch>
建立分支并切换工作区
git checkout -b <branch>
删除已被合并的分支
git branch -d <branch>
强制删除未被合并的分支
git branch -D <branch>
git branch -m <newbranch>
git branch -u <upstream>

git add

添加所有文件到暂存区，包括未追踪文件
git add -A
更新暂存区文件
git add -u
交互式添加文件到暂存区
git add -p

git reset

git reset
git reset --hard
恢复本地仓库到某次提交，并重置工作区与暂存区
git reset --hard <commit-ish>
恢复本地仓库到某次提交，不重置工作区与暂存区
git reset --soft <commit-ish>
恢复本地仓库到某次提交，重置暂存区
git reset --mixed <commit-ish>

git stash

储藏(stash)工作区相对暂存区更改的文件
git stash
储藏文件并添加描述信息
git stash save <message>
恢复最后一次储藏的文件并删除此次储存记录
git stash pop
查看储藏列表
git stash list

git grep

git grep hello
